## Setting Up Telegram

<Steps>
  <Step title="Open Web App">
    Navigate to **Settings ‚Üí Integrations** in Hyperaide
  </Step>
  
  <Step title="Select Telegram">
    Find and click the **Telegram** integration card
  </Step>
  
  <Step title="Start Chat">
    Click **Connect to Telegram** - this opens a Telegram chat with your assistant bot
  </Step>
  
  <Step title="Send /start">
    Send `/start` in the Telegram chat to activate your assistant
  </Step>
  
  <Step title="Confirm Connection">
    Your assistant sends a welcome message confirming the connection
  </Step>
</Steps>

<Tip>
  Add your assistant bot to your Telegram favorites for quick access from any device.
</Tip>

## Using Telegram Commands

Telegram supports special bot commands for quick actions:

<Tabs>
  <Tab title="Basic Commands">
    ```text
    /start - Activate your assistant
    /help - Show available commands
    /today - Show today's tasks and schedule
    /tomorrow - Show tomorrow's agenda
    /tasks - List all active tasks
    /done - Mark the last mentioned task complete
    ```
  </Tab>
  
  <Tab title="Quick Actions">
    ```text
    /add [task] - Quickly add a task
    /remind [time] [message] - Set a reminder
    /note [content] - Create a quick note
    /search [query] - Search across all your data
    ```
  </Tab>
  
  <Tab title="Information">
    ```text
    /status - Show assistant status and stats
    /integrations - List connected integrations
    /help [topic] - Get help on specific features
    /settings - Adjust Telegram-specific settings
    ```
  </Tab>
</Tabs>

<Info>
  You can also use natural language without commands‚Äîyour assistant understands both styles!
</Info>

## Troubleshooting

<AccordionGroup>
  <Accordion title="Bot Not Responding" icon="robot">
    **Solutions:**
    1. Send `/start` to reactivate
    2. Check integration status in web app
    3. Try sending from a different device
    4. Reconnect the integration
  </Accordion>
  
  <Accordion title="Commands Not Working" icon="slash">
    **Solutions:**
    1. Ensure commands start with `/`
    2. Check spelling (commands are case-sensitive)
    3. Try the natural language equivalent
    4. Send `/help` to see available commands
  </Accordion>
  
  <Accordion title="Voice Notes Not Transcribing" icon="microphone-slash">
    **Solutions:**
    1. Ensure voice note is clear
    2. Check language settings match your speech
    3. Try shorter voice notes (under 2 minutes)
    4. Send as text if transcription fails repeatedly
  </Accordion>
</AccordionGroup>
