## Setting Up Email

<Steps>
  <Step title="Find Your Address">
    Go to **Settings → Integrations**
  </Step>
  
  <Step title="Copy Email Address">
    Your unique assistant email is displayed (e.g., `assistant-abc123@hyperaide.com`)
  </Step>
  
  <Step title="Save to Contacts">
    Add this email to your contacts
  </Step>
  
  <Step title="Start Emailing">
    Send your first email to your assistant!
  </Step>
</Steps>

## What You Can Do

<Tabs>
  <Tab title="Direct Communication">
    Email your assistant with requests:
    
    ```text
    Subject: Tasks for next week
    
    Hi assistant, please create these tasks:
    1. Review Q4 budget proposal
    2. Schedule 1:1s with team members
    3. Prepare presentation for client meeting
    
    All due by Friday.
    ```
  </Tab>
  
  <Tab title="Forward Emails">
    Forward important emails for processing:
    
    ```text
    Forward email with note:
    "Save this client request to my Client Interactions database 
     and create a follow-up task for Monday"
    ```
    
    Your assistant extracts key information and takes action.
  </Tab>
  
  <Tab title="Document Processing">
    Attach files for your assistant to handle:
    
    ```text
    "Here's the budget spreadsheet. 
     Extract key numbers and add to my Finance database"
    ```
  </Tab>
  
  <Tab title="Batch Operations">
    Send multiple requests in one email:
    
    ```text
    "Please:
    1. Show my tasks for today
    2. Create a task to review the contract
    3. Add John's birthday to my contacts
    4. Schedule a reminder for the team meeting prep"
    ```
  </Tab>
</Tabs>

## Email-Specific Features

- **Subject line parsing**: Subject becomes task title if creating tasks
- **HTML formatting**: Your assistant understands formatted emails
- **Thread context**: Maintains context in email conversations
- **Reply handling**: Can continue discussions via email threads

## Best Use Cases

<AccordionGroup>
  <Accordion title="Email Forwarding" icon="share">
    Forward emails you need to act on:
    - Client requests → Tasks
    - Newsletter insights → Notes
    - Event invitations → Calendar
    - Important updates → Database records
  </Accordion>
  
  <Accordion title="Detailed Instructions" icon="list">
    When you need to provide extensive context:
    - Project specifications
    - Multi-step workflows
    - Batch task creation
    - Complex database updates
  </Accordion>
  
  <Accordion title="Document Sharing" icon="file">
    Attach files for processing:
    - PDFs for summarization
    - Spreadsheets for data extraction
    - Images for context
    - Documents for reference
  </Accordion>
</AccordionGroup>

## Tips

<CardGroup cols={2}>
  <Card title="Clear Subject Lines" icon="heading">
    Use descriptive subjects—they help your assistant understand intent
  </Card>
  
  <Card title="Forward Strategically" icon="filter">
    Forward emails that need action, not everything
  </Card>
  
  <Card title="Add Context" icon="message">
    When forwarding, add a note explaining what you want done
  </Card>
  
  <Card title="Batch Requests" icon="layer-group">
    Send multiple requests in one email for efficiency
  </Card>
</CardGroup>

<Note>
  Email is perfect for desktop users, forwarding important messages, and when you need to provide detailed context that's easier to type than speak.
</Note>
