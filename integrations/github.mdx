## Setting Up GitHub

<Steps>
  <Step title="Go to Integrations">
    Navigate to **Settings â†’ Integrations**
  </Step>
  
  <Step title="Select GitHub">
    Click the **GitHub** integration card
  </Step>
  
  <Step title="Authorize">
    Sign in with GitHub and grant access
  </Step>
  
  <Step title="Choose Repositories">
    Select which repos to sync
  </Step>
  
  <Step title="Start Tracking">
    Your assistant can now interact with GitHub!
  </Step>
</Steps>

## Smart Features

- **Label detection**: Understands bug, feature, enhancement labels
- **Assignee mentions**: "@john" assigns to team members
- **Milestone tracking**: Links issues to milestones automatically
- **Status awareness**: Knows open, closed, merged states

## Example Workflows

```text
Daily Standup:
"Show my GitHub activity from yesterday and today's planned work"

Code Review:
"Every morning, show PRs waiting for my review"

Bug Triage:
"When a critical bug is mentioned, create a GitHub issue immediately"

Release Prep:
"Before each release, show all issues in the milestone"
```

<Note>
  GitHub integration is essential for developers who want to stay on top of their repositories without constantly checking GitHub.
