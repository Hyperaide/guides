## Setting Up WhatsApp

<Steps>
  <Step title="Open Web App">
    Navigate to **Settings ‚Üí Integrations** in the Hyperaide web app
  </Step>
  
  <Step title="Select WhatsApp">
    Find and click on the **WhatsApp** integration card
  </Step>
  
  <Step title="Get Verification Code">
    You'll see a 6-digit verification code displayed on screen
  </Step>
  
  <Step title="Send Verification Code">
    Send this code as a WhatsApp message to the number provided (usually displayed as a QR code or clickable link)
  </Step>
  
  <Step title="Confirm Connection">
    Your assistant will reply with a confirmation message
  </Step>
  
  <Step title="Save Contact">
    Save your assistant's number in your contacts for easy access
  </Step>
</Steps>

<Tip>
  **Pro tip**: Save your assistant with a name like "My Assistant" or "Hyperaide" so it's easy to find in WhatsApp.
</Tip>

## Privacy & Security

<Warning>
  **Important**: WhatsApp messages are subject to WhatsApp's privacy policy. Hyperaide processes your messages securely but cannot control WhatsApp's infrastructure.
</Warning>

<AccordionGroup>
  <Accordion title="Message Security" icon="shield">
    - Messages between you and your assistant are encrypted by WhatsApp
    - Hyperaide processes messages on secure servers
    - Messages are never shared with third parties
    - You can delete your message history anytime
  </Accordion>
  
  <Accordion title="What Hyperaide Stores" icon="database">
    - Message content to maintain conversation context
    - User instructions and data you explicitly provide
    - Message metadata (timestamps, message IDs)
    - Voice note transcriptions (after processing)
  </Accordion>
  
  <Accordion title="Voice Note Processing" icon="microphone">
    - Voice notes are transcribed using secure AI services
    - Audio files are temporarily stored during transcription
    - Audio is deleted after successful transcription
    - Transcriptions are retained like text messages
  </Accordion>
</AccordionGroup>

## Troubleshooting

<AccordionGroup>
  <Accordion title="Assistant Not Responding" icon="circle-exclamation">
    **Possible solutions:**
    1. Check your internet connection
    2. Verify the integration is still active in Settings
    3. Try sending "hello" to test the connection
    4. Check if WhatsApp itself is working (try messaging someone else)
    5. Reconnect the integration if needed
  </Accordion>
  
  <Accordion title="Verification Code Not Working" icon="key">
    **Possible solutions:**
    1. Make sure you're sending to the correct number
    2. Copy the code exactly as shown (6 digits)
    3. Try generating a new code if the old one expired
    4. Check that you're using the phone number registered with Hyperaide
  </Accordion>
  
  <Accordion title="Voice Notes Not Transcribing" icon="microphone-slash">
    **Possible solutions:**
    1. Ensure the voice note is clear and not too quiet
    2. Try sending a text message to verify the connection works
    3. Check if the voice note is too long (max 5 minutes)
    4. Try recording again with less background noise
  </Accordion>
  
  <Accordion title="Messages Out of Sync" icon="arrows-rotate">
    **Possible solutions:**
    1. Wait 30 seconds (processing can take a moment)
    2. Send a new message to trigger a sync
    3. Check the web app to see if changes are reflected there
    4. Reconnect WhatsApp if the issue persists
  </Accordion>
</AccordionGroup>

## Disconnecting WhatsApp

If you need to disconnect WhatsApp:

<Steps>
  <Step title="Open Settings">
    Go to **Settings ‚Üí Integrations** in the web app
  </Step>
  
  <Step title="Find WhatsApp">
    Locate your WhatsApp integration
  </Step>
  
  <Step title="Disconnect">
    Click **Disconnect** and confirm
  </Step>
  
  <Step title="Verify">
    Your assistant will no longer respond to WhatsApp messages
  </Step>
</Steps>

<Info>
  Your conversation history and data remain in Hyperaide even after disconnecting. You can reconnect WhatsApp anytime without losing information.
</Info>

<Note>
  WhatsApp is the most popular communication channel for Hyperaide users because it's instant, familiar, and always with you. Set it up first and you'll wonder how you lived without it!
</Note>
